## 💡 Problem
Twitter'da 42 farklı uygulama geliştiren bir yazılımcının hikayesine rastladım. Projelerinin hiçbiri viral olmamıştı ama her birini sadece 2-4 sadık kullanıcı ayakta tutuyordu. Kendi kendime sordum: 

*Bu geliştirici, sadece birkaç kullanıcı için bu projeleri ne kadar süre devam ettirebilir? Eğer o son birkaç kişi de kullanmayı bırakırsa binlerce satırlık emek ve fikir sadece bir **"dijital mezarlığa"** mı dönüşecek?* Bu, hem devasa bir emeğin yok olması hem de bir topluluk ihtiyacının sahipsiz kalmasıdır.

## 🧪 Çözüm: Ozmoz
Ozmoz; geliştiricisi tarafından sürdürülemeyen ancak potansiyeli olan projelerin başkaları tarafından devralındığı, yenilendiği ve canlandırıldığı bir **Restorasyon Pazaryeri**'dir. 

**Google Gemini API**, sisteme yüklenen atıl projeleri analiz ederek teknik borç haritası ve bir **"Revival Score"** (Canlanma Skoru) çıkarır. Böylece bir geliştiricinin bırakmak zorunda kaldığı projeyi, topluluk içinden yeni eller devralarak modernizasyon sürecini başlatır. Ozmoz, sevdiğimiz uygulamaların yapımcısı bıraksa bile topluluk eliyle sonsuza dek yaşamasını sağlar.

## 🛠️ Kullanılan Teknolojiler

* **Frontend:** React 18, Vite, Tailwind CSS (Genesis Dark Theme)
* **Görsel & Animasyon:** Framer Motion, Lucide Icons
* **AI Engine:** Google Gemini API (Google AI Studio üzerinden nöral analiz)
* **Development Strategy:** Cursor AI (Vibe Coding & Agentic Workflow)
* **Deployment:** Netlify Edge (Global CI/CD & Performance Optimization)

## 🚀 Kurulum ve Çalıştırma

Projeyi yerel ortamınızda ayağa kaldırmak için aşağıdaki adımları izleyin:

1.  **Repoyu clone'layın:**
    ```bash
    git clone [https://github.com/ozmiao/ozmozgenesis](https://github.com/ozmiao/ozmozgenesis)
    cd ozmozgenesis
    ```

2.  **Gerekli paketleri yükleyin:**
    ```bash
    npm install
    ```

3.  **Çevre değişkenlerini ayarlayın:**
    Ana dizinde bir `.env` dosyası oluşturun ve Gemini API anahtarınızı ekleyin (Bkz: `.env.example`):
    ```env
    VITE_GEMINI_API_KEY=Kendi_API_Anahtarınızı_Buraya_Yazın
    ```

4.  **Uygulamayı başlatın:**
    ```bash
    npm run dev
    ```

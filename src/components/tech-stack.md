# 🛠 OZMOZ Technical Architecture

## Frontend & UI
- **Framework:** React 18 with Vite.
- **Styling:** Tailwind CSS (Genesis Dark Theme).
- **Animations:** Framer Motion (Used for "neural links", smooth transitions, and live tickers).
- **Icons:** Lucide React (Cyber-archeology iconography).

## AI Integration
- **Model:** Google Gemini API via Google AI Studio.
- **Functionality:** Used for the "Neural Analyze" feature which provides architectural audits and strategic migration pathways for legacy code.

## Development & Deployment
- **Editor:** Cursor AI (Used for Vibe Coding and Agent-based refactoring).
- **Version Control:** GitHub.
- **Deployment:** Lovable.dev / Netlify.
- **Telemetry Simulation:** Real-time state-based updates for CPU load, latency, and node discovery.
